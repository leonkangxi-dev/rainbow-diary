### 3. Bug 修复

#### Bug 1：语音开关无法保存 + 编辑页没有录音按钮
- **根因**：`sql.js` 命名参数绑定要求 key 带 `@` 前缀，但 `updateSettings` 中写的是 `params[key]` 而非 `params['@' + key]`
- **修复文件**：
  - `src/js/database.js:95` — `params['@${key}'] = val`
  - `server.js:121` — 同上
- **效果**：语音开关切 ON 后真正写入 DB，退出再进不会变灰，编辑页可看到录音按钮

#### Bug 2：孩子 PIN 无法修改
- **根因**：家长设置页孩子列表只有"删除"按钮，缺少修改密码的 UI
- **修复**：将修改密码功能移到孩子登录后的书架页面
  - `src/index.html` — 书架 header 加"🔑 修改密码"按钮 + 弹窗模态框
  - `src/js/app.js` — 新增 `childSettings` 对象（`showPinChange` / `hidePinChange` / `confirmPinChange`）
  - 流程：输入旧密码验证 → 输入新密码 → 调 `updateUserPin` 写入 DB

#### 附带清理：移除录音文件保存逻辑
- 用户需要的是语音转文字（speech-to-text），而非录音存储
- 简化 `onstop` 为仅清理 stream
- 简化 `editor.save()` 中 `audio_path` 为 `''`
- 保留 Web Speech API 语音识别功能
