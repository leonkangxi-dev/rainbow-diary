# 🌈 彩虹日记本 (Rainbow Diary)

> 专为小学生设计的可爱风格日记软件 ✨

![版本](https://img.shields.io/badge/version-1.0.0-FFB7C5)

---

## 🌟 特性

- 🎀 **三丽鸥角色封面** — 7 款可爱角色可选，每款对应主题配色
- 👨‍👩‍👧‍👧 **多用户支持** — 家庭多个孩子各自独立使用
- 🔒 **日记本加密** — 每个日记本可独立设置 4 位数字密码
- 🎤 **语音输入** — 支持语音转文字（需家长开启）
- 🏆 **成就系统** — 连续写日记获得成就徽章
- 🌐 **双模式** — 本地桌面 + 局域网多设备访问

## 🚀 快速开始

### 📦 便携版（推荐）

1. 从 `dist/win-unpacked/` 解压
2. 双击 `彩虹日记本.exe`
3. 开始写日记吧！

### 🖥 开发模式

```bash
npm install
npm start
```

### 🌐 局域网访问

```bash
node server.js
```

浏览器访问 `http://你的IP:3000`（首次需 `添加防火墙规则.bat`）

## 📖 首次使用

| 步骤 | 操作 |
|------|------|
| 1 | 点击首页右下角 🔒 **家长设置** |
| 2 | 输入密码 `1234`（可修改）|
| 3 | 在 **添加孩子** 中创建用户 |
| 4 | 返回首页选择孩子，输入 PIN（默认 `1234`）|
| 5 | 开始写日记！🎉 |

## 🛠 技术栈

- **框架：** Electron 33
- **数据库：** SQLite (sql.js)
- **后端：** Express 4
- **语音：** Web Speech API
- **打包：** electron-builder
