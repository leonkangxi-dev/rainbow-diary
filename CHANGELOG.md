# 更新日志

## v1.2026.517 (2026-05-17)

### 🎤 语音输入改用讯飞 API
- 移除 Google Web Speech API（国内不可用）
- 集成讯飞语音听写（IAT）WebAPI v3
- 家长设置中添加 APPID / API Key / API Secret 配置
- 录音方式改为 16kHz 16bit PCM 音频捕获
- 识别过程改为录制完成后发送识别

### 🌐 局域网服务改进
- 「启动局域网服务」开关现在真正启动 Express 服务（之前仅为 UI 提示）
- 服务与桌面版共享同一个数据库，数据完全一致
- 端口占用时自动向后寻找可用端口
- 界面显示实际可访问的本机/局域网地址

### 🖼️ 应用图标修复
- BrowserWindow 设置自定义图标（彩虹图案）
- 添加 `npm run generate-icon` 脚本

### 🐛 其他修复
- Electron `file://` 安全上下文问题 → 改用 `app://` 协议
- 语音识别错误信息更详细（显示具体错误码）
- 设置页新增讯飞凭据 UI
